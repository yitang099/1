WeGameTQ 验证码助手（纯本地版）
================================

不需要图灵账号！全部本地识别。

支持：
  1. 选图（兔子、浅紫色手表等）→ OCR读题 + 中文CLIP匹配图片
  2. 动态点选（运动最慢的元素）→ OpenCV追踪圆点

不支持：滑动滑块（靠 WeGameTQ 自带打码）

安装（首次）
------------
1. 安装 Python 3.10+（勾选 Add to PATH）
2. 解压 captcha_auto 文件夹
3. 打开 cmd，进入目录：
   cd C:\Users\Administrator\Desktop\captcha_auto
4. 安装依赖：
   python -m pip install -r requirements.txt
5. 下载 AI 模型（约400MB，只需一次）：
   双击 install_models.bat

使用
----
1. 打开 WeGameTQ，等验证码出来
2. 双击 calibrate.bat → 框选整个验证码（含顶部题目+6张图+确定按钮）
3. 双击 start_silent.bat → 后台自动过码
   或 start_background.bat → 有窗口看日志
   或 start_manual.bat → 手动按 F8/F9/F10

手动按键：
  F8  = 自动判断题型
  F9  = 强制选图（本地CLIP）
  F10 = 强制动态点选
  ESC = 退出

日志：captcha_auto.log
失败截图：debug 文件夹

注意
----
1. 校准区域必须包含顶部题目文字，否则 OCR 读不到题
2. 置信度不够时会跳过，不会乱点（比瞎点好）
3. 第一次识别较慢（加载模型），之后就快了
4. 滑块题仍需软件自带打码

若中文 bat 报错，用英文名：
  calibrate.bat / start_silent.bat / install_models.bat
