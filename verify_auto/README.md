# 两步验证助手 v0.2

同一窗口两步验证（如你发的截图）：

1. **按字选图** — `选择最符合描述的图片: '兔子'` → 点对应图 → 确定  
2. **最慢动球** — 很多**不动**的装饰球 + 少数**会动**的球 → 在**动球里**选最慢的 → 确定  

## 安装

```bash
cd verify_auto
pip install -r requirements.txt
# 第1步识图更准（可选，需下载模型）：
# pip install torch transformers
python -m verify_auto
```

## 首次校准（各框一次）

| 按钮 | 框什么 |
|------|--------|
| 提示文字区 | 顶部「选择最符合描述的图片…」 |
| 图片网格区 | 中间 2×3 六张图（不要包确定） |
| 第2步球区域 | 第二步所有球所在区域 |
| 确定按钮 | 蓝色「确定」 |

然后 **F8 一键全自动**。

## 第2步原理

- 连续截多帧，用**帧差分**找会动的斑点  
- 总位移 < 6px 的当作**不动装饰球**，忽略  
- 只在**动球**里选位移最小的 = 最慢的那个  

## 第1步原理

- OCR 读引号里的词（如「兔子」）  
- 把网格切成 6 格，用 Chinese-CLIP 语义匹配（没装 torch 时用弱兜底，建议装）  

OCR 读不到字时，在界面填 **关键词** 手动指定。
