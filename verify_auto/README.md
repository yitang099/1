# 两步验证助手

你的验证**不是滑块**，是：

1. **按字选图** — 看提示字，点对应图片  
2. **点最慢的球** — 多个彩色球在动，找最慢的 → 点它 → 点确定

## 第2步用法

1. 弹出「选最慢球」时，点 **框选第2步球区域**（只框球，不要包确定按钮）
2. 点 **框选确定按钮**
3. **找最慢球并点击** — 会自动采样约 1～2 秒、点最慢的球、再点确定
4. 不准就先点 **只分析不点击** 看日志里各球移动量
