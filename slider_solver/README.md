# 滑块学习复现（桌面软件）

**你要的模式：** 先手动拖过一次 → 自动保存 → 以后遇到**相同验证图**自动拖。

## 安装（Windows）

```bash
cd slider_solver
pip install -r requirements.txt
python -m slider_solver
```

## 使用

| 步骤 | 操作 |
|------|------|
| 1 | 打开你的软件，弹出滑块 |
| 2 | 本工具点 **框选区域** → 拖框盖住整条滑块 |
| 3 | 点 **开始录制 (F9)** |
| 4 | 去你的软件里 **手动拖一次** 滑块（在框内按下→拖到终点→松开） |
| 5 | 工具自动保存验证截图 + 拖动距离到 `records/` |
| 6 | 下次相同图出现 → 按 **F8** 或开 **后台监听** 自动过 |

## 快捷键

- **F9** 开始录制（然后手动拖一次）
- **F8** 立即匹配并自动拖动

## 参数

| 参数 | 说明 |
|------|------|
| 匹配阈值 | 默认 0.88，越高越要求图像几乎一样 |
| X微调 | 自动拖时 ± 几像素 |
| 监听间隔 | 后台扫描间隔（毫秒） |

## 原理

录制时保存：
- 验证码区域截图（`captcha.png`）
- 滑块按钮小图（`knob.png`）
- 你拖动的横向距离（像素）

自动时：
- 截取当前区域图，与库内记录比对
- 相似度 ≥ 阈值 → 找到滑块按钮 → 按保存的距离拖动

## 限制

- 只适用于 **验证图完全一样**（或几乎一样）的情况
- 若每次缺口位置都变，需要每条图各录制一次，或改用识别缺口模式

## 合规

仅用于你有权本地测试的软件。
