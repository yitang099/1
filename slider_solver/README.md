# 滑块自动拖动（桌面软件）

针对 **Windows 上已有软件** 里的滑块验证：自动找滑块按钮 → 识别距离 → 模拟鼠标拖动。

## 安装（Windows）

```bash
cd slider_solver
pip install -r requirements.txt
python -m slider_solver
```

## 使用步骤

1. 打开你的软件，弹出滑块验证
2. 运行本工具
3. **框选验证码区域**（盖住整条滑块条）
4. **截取滑块按钮**（只框左边可拖动的小块）
5. 按 **F8** 或点 **试跑一次**

## 调参

| 参数 | 说明 |
|------|------|
| 固定距离 | 若自动识别不准，填实测像素距离（0=自动） |
| X微调 | ±几像素微调 |
| 拖动耗时 | 越大越慢，更像真人 |
