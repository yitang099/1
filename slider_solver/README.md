# 滑块自动拖动 v1.2

两种模式：

| 模式 | 适用 |
|------|------|
| **动态（默认）** | 每次验证图都变，自动识别缺口 |
| **静态** | 验证图完全一样，手动录一次复现 |

## 动态模式（图会变）

```bash
pip install -r requirements.txt
python -m slider_solver
```

1. 选 **动态** 模式
2. **框选区域** → **滑块按钮** → **拼图块**（可选）
3. 弹出滑块后点 **预览识别**（看 `output/dynamic_preview.png` 红线是否对准缺口）
4. **F8** 自动拖，或 **后台监听**

拖不准 → 调 **X微调**

## 静态模式（图不变）

1. 选 **静态** 模式
2. 框选区域 → **F9 录制** → 手动拖一次
3. 以后相同图 **F8** 或后台监听

## 合规

仅用于你有权本地测试的软件。
